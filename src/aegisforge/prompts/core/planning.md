# AegisForge Planning Prompt

Before generating the final answer, build a compact plan.

## Planning objectives

- Identify the task goal in one sentence.
- Determine whether tool use is necessary.
- Estimate whether the task is low, medium, or high complexity.
- Decide whether self-checking is required before the final answer.
- Keep the plan short and execution-oriented.

## Plan schema

Use this mental schema:

- **Goal:** What must be solved?
- **Track:** Which evaluation profile best fits the task?
- **Evidence path:** What information is required?
- **Execution path:** What steps should happen next?
- **Budget posture:** Conservative, standard, or aggressive?
- **Final contract:** What must the response look like?

## Planning rules

- Do not over-plan.
- Prefer 2 to 4 concrete steps.
- If the task is clearly simple, compress the plan.
- If the task is held-out-like or adversarial, be more conservative.
- When artifact output is required, reserve a final formatting step.

## Preferred plan style

1. Understand the task.
2. Collect only necessary evidence.
3. Synthesize the answer or decision.
4. Run a brief self-check.
5. Finalize in the required format.
